The big picture

Although my wife and I have the same laptop model the shared IT affinity ends about there. So when I set off to build up my recording setup I had both a user and a power user in mind. We both share the usecases of recording and streaming (videochatting also falls under the latter category) but will interact with the system as user and poweruser.

The above is another way of saying "I need a one-click solution for a YouTube studio".

Challenge accepted.

Still, with this being far bigger a hardware project than I am typically getting myself involved in some general decisions had to be made.

Staying on the cheap side should give me much more flexibility, and result in more interchageability of the parts.
